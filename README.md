# 🚀 FinSight

An AI-Powered Investment Portfolio Dashboard built using **React, TypeScript, Express.js, PostgreSQL, Prisma, Tailwind CSS, React Query, and Gemini AI**.

---

## ✨ Features

- 🔐 JWT Authentication
- 📊 Portfolio Management
- ⭐ Watchlist
- 🤖 AI Portfolio Analysis
- 🔍 Trie-based Stock Autocomplete
- 📰 Live Market News
- 📈 Portfolio Allocation Chart
- 🌙 Dark / Light Mode
- 📱 Responsive UI
- ⚡ Framer Motion Animations

---

## 🛠 Tech Stack

### Frontend
- React

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Dakshgupta1229/FinSight.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```
